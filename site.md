## clip

### step 1
1. instalar github-pages

- criar ficheiro Gemfile com
  gem 'github-pages'
  source 'https://rubygems.org'
- executar:
  bundle install

2. Instalar o jekyll
- executar:
  bundle exec jekyll new home

3. Iniciar repositório
- executar:
cd home
git init
